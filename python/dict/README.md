dict
====

linux命令行下翻译工具

---
#### Install

```
git clone https://github.com/jianvhen/dict.git
mv ./dict.py /usr/bin/dict
chmod +x /usr/bin/dict
```

#### Usage

```
$ dict test
###################################
#  test 测试 (U: tɛst E: test )
#  n. 试验；检验
#  vt. 试验；测试
#  vi. 试验；测试
#  n. (Test)人名；(英)特斯特
###################################
```
